#proiect DAW
Farcas Marius Angel-Informatica An3 Grupa-1

Proiectul meu este despre un subbrand auto denumit AMG in care poti cauta informatii specifice despre diferite modele de masini care poarta numele AMG, site ul este alcatuit din mai multe fisiere html, css si un fisier javascript in care am adaugat functionalitatea form ului pentru contact, funtia de traducere site si o modalitate de filter cars atunci cand cautam un anumit model de masina, site ul este format din mai multe pagini precum:

-Home - care contine un news unde se regaseste un buton care va duce catre informatii mai detaliate

-What is amg? - unde se gasesc informatii despre divizia brand ului mercedes numita AMG

-Car List - este pagina in care cauti masina dorita si in urma apasarii tag ului cu denumirea masinii cautate vei fi redirectionat catre pagina cu specificatiile masinii

-Buy an AMG - cuprinde un link care duce catre site ul oficial mercedes AMG direct in configurator de unde poti achizitiona modelul dorit

-Contact - sectiunea contact cuprinde un form cu mai multe campuri unde introduci datele personale si mesajul dorit, care functioneaza bineinteles!

Site ul mai beneficiaza si de un language selector oferind 4 limbi disponibile la alegere

Site ul poate fi accesat prin introducerea acestui link in browser: mariusangel.github.io
